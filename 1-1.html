Les constantes

 Définition :

Un constant est un objet dont la valeur reste fixe durant l’exécution d’un programme.

Déclaration en Pascal:

En Pascal, on déclare une constante comme suit :

Const nom = valeur de la constante ;

Const Pi = 3.14 ;

Les Variables

 Définition :

Une variable est un nom qui sert à repérer un emplacement donné de la mémoire centrale dont on peut faire évoluer la valeur au fil de déroulement du programme.

Déclaration en Pascal :

En Pascal, on déclare une variable comme suit :

Var nom : type ;

Var K : real ;
