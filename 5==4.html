Sur un réseau local, chacune machine est caractèrisée par son adresse IP sous la forme suivante : W.X.Y.Z

Exemple : 192.10.6.2 (01100000,00001010,00000110,00000010)

Selon la valeur prise par w, les adresses IP peuvent être classées en 3 classes:

Classe A : La valeur de A comprit entre 0 et 126

Classe B : La valeur de B comprit entre 128 et 191

Classe C : La valeur de C comprit entre 192 et 223

Remarque : N'oubliez pas les adresses reservées

On vous demande d'écrire un programme permet de :

    * Remplir aléatoirement une matrice M de N lignes et 32 colonnes par 0 et 1.

    * Former à partir de la matrice M quatre tableaux contenant chacun les adresses par classe. Ces adresses IP enregistrées dans les tableaux doivent être écrites à la base décimale sous la forme W.X.Y.Z
