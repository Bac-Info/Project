Les procédures

Fonction
Procedure

Précédent
Aide

Une procédure est un sous-programme qui retourne un ou plusieurs résultats de différents types.

 Appel :
En Pascal :

Nom_Procedure (PE 1, PE 2 .. PEn)

Analyse/Algo : :

Proc Nom_de_procédure (PE1,PE2, .. PEn)

Déclaration :
En Pascal(PF : paramètre formel) :

Procedure nom_de_procédure(paramètres);

Var definition;

Begin

Instructions de la procédure;

end;

Analyse :

Def proc nom(PFn:type n)

Résultat

liste de déf

Fin nom

Algorithme :

0) Déf proc nom(PF1,PF2, .. :Type1 ; PF1,PF2, ..:Type2; ..)

1) instructions de la procédure

2) Fin nom
