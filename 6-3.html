Définition:


Convertir un entier positif N dans une base b au moins égale à 2 consiste à chercher la représentation de N dans la base b. 
En effet, dans un système de numération b, tout entier positif N admet une et une seule décomposition : 
N = anbn + an-1bn-1 + an-2bn-2 + … + a1b + a0 où les ak sont des naturels compris entre 0 et (b-1) inclus. b est appelée base de numération.
