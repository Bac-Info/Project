Présentation :

On est souvent incité à appeler une fonction à partir d’elle-même.

Cette façon récurrente de faire permet non seulement de définir des fonctions mais aussi des ensembles et des suites.

Pour définir une fonction sur l’ensemble des entiers positifs on procède de la manière suivante:

1/ On précise la valeur de la fonction en zéro (ou en 1)

2/ On donne la règle pour trouver la valeur de la fonction à partir de sa ou de ses valeurs précédentes. Cette façon récurrente de faire appelée récursivité est une conséquence de l’induction mathématique (preuve par induction).

Les exemples ci-dessous clarifient ce concept.

CALCUL RÉCURSIF D'UN FACTORIEL : 

Function Fact(n : integer) : longint ; 


 Begin 


	 If ( n = 0 ) then


		Fact := 0


	 Else


		Fact := n * fact(n-1);


 End;

CALCULER LA FONCTION PUISSANCE : 

Function Puiss( x : Real ; n : integer ) : integer ;


 Begin


  If ( n = 0 ) then


   Puiss := 1 


  Else


  If ( n > 0 ) then


   Puiss := x * Puiss(x,n-1) ;


  Else


   Puiss := Puiss(1/x,-n) ;


 End;

Verifier Si une chaine est palindrome :

est une figure de style désignant un texte ou un mot dont l'ordre des lettres reste le même qu'on le lise de gauche à droite ou de droite à gauche, comme dans la phrase « Ésope reste ici et se repose » ou encore « La mariée ira mal » à un accent près.

Function Verif( Ch : String ) : Boolean ;


 Begin


		If ( length(ch) = 1 ) then


		 Verif := True


		Else 


		If (ch[1] <> ch[length(ch)]) then


		  Verif := False


		Else


		 Begin


		Delete(ch,1,1);


		Delete(ch,length(ch),1);


		Verif := Verif(ch) ;


		 End;


 End;


Le tour de Hanoy :

Le tour de Hanoy est un jeu qui consistre à déplacer N disques superposés l'un sur l'autre du plus grand au plus petit 
d'un piquet d départ D vers un piquet d'arriver A par l'intermediaire I.


Pour déplacer ces disques il faut respecter les règles suivantes:


    

    - On ne peut pas déplacer qu'un seul disque à la fois.
    

    - On déplace un disque vers un autre qui est plus grand.
    

    ------ Remarque : Pour déplacer N disques il faut (2^n - 1) déplacements.
