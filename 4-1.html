Le recherche sequencielle 


Traduction en Pascal :

Function Rech_Seq( t : tab ; x,n : integer ): boolean ;
var i : integer ;
    b : boolean ;
begin
 i := 0 ;
 b := false ;
 repeat
  i := i + 1 ;
  if t[i] = x then
   b := true ;
 until (b) or ( i=n );
Rech_Seq := b ;
end;




Le recherche Dichotomique 


Définition :

Le tableau doit etre trié.
On regarde l'élément au milieu du tableau. S'il est égale à la valeur cherchée, donc c'est terminé. S'il est inférieure à la valeur cherchée, 
alors il ne reste à traité que la moitié droite du tableau. S'il est supèrieure à la valeur cherchée, alors il ne reste à traité que la moitié gauche du tableau.
=> On continue la recherche en diminuant à chaque fois de moitié le nombre du tableau restant à traiter.

Traduction en Pascal :

Function Rech_Dich( t : tab ; x,n : integer ): boolean ;
var d,f,i,m : integer ;
    b : boolean ;
begin
 f := n ;
 d := 1 ;
 b := false ;
 repeat
  m := (d+f) div 2 ;
  if ( t[m]= x ) then
   b := true
  else if (t[m] < n ) then
   d := m + 1 
  else
   f := m - 1 ;
 Until (d > f ) or (b) ;
 Rech_Dich := b ;
end;
